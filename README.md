Rust Drivers for HDL cores
=======

Collection of native Rust driver for various HDL cores.

Currently contains drivers for the following HDL blocks:

- [`axi-uartlite`](./axi-uartlite/) for the [AXI UARTLITE](https://www.amd.com/es/products/adaptive-socs-and-fpgas/intellectual-property/axi_uartlite.html)
- [`axi-uart16550`](./axi-uart16550/) for the [AXI UART16550](https://www.amd.com/en/products/adaptive-socs-and-fpgas/intellectual-property/axi_uart16550.html)
- [`axi-dma`](./axi-dma/) for the [AXI DMA Controller](https://www.amd.com/en/products/adaptive-socs-and-fpgas/intellectual-property/axi_dma.html)
- [`axi-ad9361`](./axi-ad9361/) for the the Analog Devices [AXI AD9361 core](https://analogdevicesinc.github.io/hdl/library/axi_ad9361/index.html)
